Hallo!

MAG2DATA is an open source, 2D and 2-1/2D magnetic data processing
software package.

MAG2DATA runs in MATLAB 7.0 and 2008a.

Read HELP.pdf for using it.

--------------------------------------------------------------------------
Installation
--------------------------------------------------------------------------
Copy all the *.m files in a new directory and make sure that the "current
directory" in your Matlab environment indicates the new directory where
you put the *.m files.
Type in MAG2D in the Matlab command window for running the program.
